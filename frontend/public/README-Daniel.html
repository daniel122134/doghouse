Additional Features

Our social network includes the following extra features:

  • Profile picture
  • Share post by email button
  • Unbark (unlike)
  • Search users by substring
  • Peeing on poles on PeePoles page
  • Edit Post Content
  • Swagger documentation

Challenges

  • Implementing user profile pictures, including handling image uploads and displaying them on user profiles
  • Implementing the Authentication Mechanism properly and securely
  • Learning new frameworks such as mocha was quite challenging but rewarding

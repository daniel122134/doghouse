Deployment Instructions

inside our project we have 3 package.json files
  • frontend/package.json - contains the frontend dependencies and scripts
  • backend/package.json - contains the backend dependencies and scripts
  • package.json - contains the deployment scripts
  • the first two correspond to the frontend and backend respectively.
  • the third is used for deployment purposes. it contains the following scripts:
    • npm run test - runs the backend api tests against te localhost server at port 8080
    • npm run all-prod - starts the backend server and builds the frontend at the same time
    • npm run server-prod - runs only the backend - needs to be run before running the tests
